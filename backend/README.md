# Smart Farmers Backend API

## Overview
Node.js + Express + MongoDB REST API for the Smart Farmers platform.

## Features
- JWT Authentication with OTP verification
- Multi-vendor marketplace management
- Product CRUD with variations
- Order management with multi-vendor splitting
- Delivery tracking with real-time location
- Consultant and financial services
- Admin dashboard with analytics
- Notification system with Socket.io
